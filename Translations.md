# Translations

This add-in uses JSON locale files in `src/NcTalkOutlookAddIn/Resources/_locales/**/messages.json`.

## Available locales

| Locale folder | Language |
|---|---|
| `de` | Deutsch (default) |
| `en` | English |
| `fr` | Français |
| `cs` | Čeština |
| `es` | Español |
| `hu` | Magyar |
| `it` | Italiano |
| `ja` | 日本語 |
| `nl` | Nederlands |
| `pl` | Polski |
| `pt_BR` | Português (Brasil) |
| `pt_PT` | Português (Portugal) |
| `ru` | Русский |
| `zh_CN` | 简体中文 |
| `zh_TW` | 繁體中文 |
